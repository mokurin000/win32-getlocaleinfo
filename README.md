# win32-locale-info

[RFC1766]: https://datatracker.ietf.org/doc/html/rfc1766
[BCP-47]: https://datatracker.ietf.org/doc/html/rfc5646

Get user locale name in [RFC 1766][RFC1766] / [BCP-47] format on Windows with ease.

Note: prefer full BCP-47 format when you need to support font localization yourself.
